# Testistic  Model

This module describes the core Testistic model.

## Testruns
Firstly it provides a model for TestRun.

TestRun needs to include
- a project
- an epic
- test source
- test status  (1 means the test ran successfully, 0 means otherwise)
- test start and end datetime
